# Виртуальный DOM

Виртуальный DOM (Virtual DOM) — это ключевая концепция React, которая делает
обновления интерфейса быстрыми и эффективными.

## Проблема реального DOM

Каждое изменение реального DOM — дорогая операция. Браузер пересчитывает стили,
перестраивает дерево элементов и перерисовывает страницу.

## Как работает Virtual DOM

1. React хранит лёгкую копию DOM-дерева в памяти — **Virtual DOM**
2. При изменении состояния создаётся новый Virtual DOM
3. React сравнивает новый Virtual DOM со старым (через **diffing**)
4. Вычисляется минимальный набор изменений для реального DOM (**reconciliation**)
5. React применяет только эти изменения

## Преимущества

- **Быстродействие** — минимум операций с реальным DOM
- **Декларативность** — вы описываете конечное состояние, а React сам решает, как до него добраться
- **Кроссплатформенность** — Virtual DOM можно рендерить не только в браузер, но и в мобильные приложения (React Native)
