# What is React?

React — это библиотека JavaScript для построения пользовательских интерфейсов.

## Ключевые особенности

- **Компонентный подход** — UI разбивается на независимые, переиспользуемые компоненты
- **Декларативность** — вы описываете, как UI должен выглядеть для каждого состояния
- **Виртуальный DOM** — React обновляет только изменившиеся части реального DOM

## Пример компонента

```tsx
function Welcome() {
  return <h1>Hello, React!</h1>;
}
```

React был создан командой Meta (Facebook) и впервые выпущен в 2013 году.
