# useEffect Dependencies

Массив зависимостей определяет, когда эффект будет перезапущен.

## Без зависимостей

```tsx
useEffect(() => {
  console.log('Выполняется при каждом рендере')
})
```

## Пустой массив

```tsx
useEffect(() => {
  console.log('Выполняется один раз после монтирования')
}, [])
```

## Конкретные зависимости

```tsx
useEffect(() => {
  document.title = `Привет, ${name}`
}, [name]) // перезапускается при изменении name
```

## Правила

1. Все реактивные значения должны быть в массиве зависимостей
2. Не врать о зависимостях — это приводит к багам
3. ESLint плагин `eslint-plugin-react-hooks` поможет отследить пропущенные зависимости
